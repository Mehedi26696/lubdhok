## About

A modern web portal for the **Lubdhok** batch (CSE 29th Batch, Dhaka University) to share study materials, academic projects, events, and announcements. Built with Next.js 15, TypeScript, and Tailwind CSS.

## Features

- Study materials organized by semester and course
- Academic project showcase with GitHub integration
- Event management (academic, social, cultural, technical)
- Batch announcements and updates
- AI-powered chatbot (Google Gemini, OpenAI, Groq)

## Tech Stack

- **Framework:** Next.js 15.4 (App Router, Turbopack)
- **Language:** TypeScript 5.x
- **Styling:** Tailwind CSS 4.x
- **UI Library:** React 19.1
- **AI Integration:** Google Generative AI, OpenAI, Groq SDK

## Getting Started

### Prerequisites

- Node.js 18+
- npm/yarn/pnpm/bun

### Installation

```bash
# Clone the repository
git clone https://github.com/Mehedi26696/lubdhok.git
cd lubdhok

# Install dependencies
npm install

# Set up environment variables (create .env.local)
GOOGLE_GENERATIVE_AI_API_KEY=your_key
OPENAI_API_KEY=your_key
GROQ_API_KEY=your_key

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

```
src/
├── app/              # Next.js pages (about, events, projects, semesters, etc.)
├── components/       # React components (Header, Footer, ChatBot, Cards)
└── data/             # Static data (projects, events, study materials)
```
